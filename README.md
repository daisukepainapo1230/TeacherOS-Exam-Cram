# TeacherOS-Exam-Cram
東京都の小学校教員採用試験に向けて、「小学校全科」と「論文」を短時間で学習する静的Webアプリです。iPhone・iPadのSafariを優先して設計しています。小学校全科はタップで解ける4択問題、論文練習はテーマ・キーワード・実践例を選ぶテンプレート式です。  小学校全科は、東京都公立学校教員採用候補者選考の公式過去問PDFの出題形式・出題分野・難易度を参考にしたオリジナル類題です。過去問本文の丸写しではありません。初期データとして、国語、社会、算数、理科、生活、音楽、図画工作、家庭、体育、外国語、特別の教科 道徳、総合的な学習の時間、特別活動の13教科・各5問、合計65問を収録しています。  学習記録画面から、問題データを次のCSV形式でコピーまたはダウンロードできます。
